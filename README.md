# Percento Tracker — Google Sheets AppScript

自動同步資產追蹤試算表的 Google Apps Script，涵蓋匯率更新與投資組合股價抓取。

## 功能

- **匯率更新**（設定工作表）：自動透過 `GOOGLEFINANCE` 抓取各幣別對新台幣匯率，失敗時保留上次成功值
- **股價更新**（投資工作表）：支援台股（TPE:xxxx）與美股 ticker，自動更新現價與時間戳記；失敗時保留原數值
- **現金帳戶時間戳記**（現金帳戶工作表）：同步更新每筆帳戶的最後更新時間
- **容錯機制**：所有欄位在資料拉取失敗時均保留舊值，不覆蓋為空白

## 快速開始

### Step 1：將 Excel 匯入 Google Sheets

1. 前往 [Google Drive](https://drive.google.com)
2. 點選左上角 **「+ 新增」→「檔案上傳」**，選擇 `Percento_Example.xlsx`
3. 上傳完成後，在 Drive 中對該檔案按右鍵 →「以 Google 試算表開啟」
4. Google Sheets 會自動轉換並開啟，確認工作表分頁包含：**總覽、設定、投資、現金帳戶**

### Step 2：開啟並貼上 AppScript

1. 在 Google Sheets 上方選單點選 **「擴充功能」→「Apps Script」**
2. 系統會開啟 Apps Script 編輯器（新分頁）
3. 刪除編輯器中預設的空白內容
4. 將本專案 `appscript` 檔案的全部內容複製貼上
5. 按 **Ctrl+S**（Mac：Cmd+S）儲存，左上角專案名稱可改為 `Percento Tracker`

### Step 3：填寫資料

回到 Google Sheets，依照以下欄位填入你的資產資料：

**設定工作表（匯率）**

| 欄位 | 說明 | 範例 |
|------|------|------|
| B欄（第4列起）| 幣別代碼 | `USD`、`JPY`、`TWD` |

**投資工作表（股票/ETF）**

| 欄位 | 說明 | 範例 |
|------|------|------|
| C欄（第2列起）| 股票代碼 | `2330`（台股）、`AAPL`、`VOO` |

> 台股只需填純數字，腳本會自動加上 `TPE:` 前綴。

**現金帳戶工作表**

| 欄位 | 說明 |
|------|------|
| A欄（第3列起）| 帳戶名稱（有填寫即會更新時間戳記）|

### Step 4：手動執行更新

1. 回到 Apps Script 編輯器
2. 確認上方函數選單顯示 **`updatePercentoTracker`**
3. 點選 **▶ 執行** 按鈕
4. 首次執行會要求授權，點選「查看權限」→ 選擇你的 Google 帳號 → 點選「允許」
5. 等待約 10 秒後，試算表彈出「✅ 同步完成！」表示成功

### Step 5：設定定時自動觸發（選用）

1. 在 Apps Script 編輯器左側點選 **⏰ 觸發條件**（鬧鐘圖示）
2. 右下角點選 **「+ 新增觸發條件」**
3. 設定如下：
   - 執行函數：`updatePercentoTracker`
   - 活動來源：`時間驅動`
   - 時間型觸發條件類型：`小時計時器` 或 `日計時器`（依需求選擇）
4. 儲存後即可自動定時更新

## 工作表結構

| 工作表名稱 | 說明 |
|-----------|------|
| 總覽 | 顯示計價幣別與最後更新時間 |
| 設定 | B欄填幣別代碼（如 USD、JPY），C欄自動更新匯率，D欄顯示更新時間 |
| 投資 | C欄填 ticker（台股純數字自動加 TPE: 前綴），F欄更新現價，L欄更新時間 |
| 現金帳戶 | A欄有內容的列，G欄自動更新時間戳記 |

## 注意事項

- 腳本使用 Google Sheets 內建的 `GOOGLEFINANCE()` 函數，**不需要任何外部 API 金鑰**
- 台股代碼請填純數字（例如 `2330`），腳本會自動補上 `TPE:` 前綴
- 美股/ETF 直接填 ticker（例如 `AAPL`、`VOO`）
- 腳本執行時間約 10 秒（含等待 Google Finance 回傳），請勿重複觸發
