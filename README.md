# Percento Tracker — Google Sheets AppScript

自動同步資產追蹤試算表的 Google Apps Script，涵蓋匯率更新與投資組合股價抓取。

## 功能

- **匯率更新**（設定工作表）：自動透過 `GOOGLEFINANCE` 抓取各幣別對新台幣匯率，失敗時保留上次成功值
- **股價更新**（投資工作表）：支援台股（TPE:xxxx）與美股 ticker，自動更新現價與時間戳記；失敗時保留原數值
- **現金帳戶時間戳記**（現金帳戶工作表）：同步更新每筆帳戶的最後更新時間
- **容錯機制**：所有欄位在資料拉取失敗時均保留舊值，不覆蓋為空白

## 使用方式

1. 開啟你的 Google Sheets 試算表
2. 點選選單 **擴充功能 → Apps Script**
3. 貼上 `appscript` 檔案內容
4. 儲存後，執行 `updatePercentoTracker()` 函數
5. 可設定觸發條件（Triggers）讓腳本定時自動執行

## 工作表結構

| 工作表名稱 | 說明 |
|-----------|------|
| 總覽 | 顯示計價幣別與最後更新時間 |
| 設定 | B欄填幣別代碼（如 USD、JPY），C欄自動更新匯率，D欄顯示更新時間 |
| 投資 | C欄填 ticker（台股純數字自動加 TPE: 前綴），F欄更新現價，L欄更新時間 |
| 現金帳戶 | A欄有內容的列，G欄自動更新時間戳記 |

## 注意事項

- 腳本使用 Google Sheets 內建的 `GOOGLEFINANCE()` 函數，**不需要任何外部 API 金鑰**
- 台股代碼請填純數字（例如 `2330`），腳本會自動補上 `TPE:` 前綴
- 美股/ETF 直接填 ticker（例如 `AAPL`、`VOO`）
- 腳本執行時間約 10 秒（含等待 Google Finance 回傳），請勿重複觸發
